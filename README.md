# 🤝 FriendKeeper

**個人社交關係管理系統** — 拍照就能記錄你和朋友的每一次相遇。

## 功能特色

- 📸 **照片自動辨識**：傳照片給 Telegram Bot，自動辨識是誰、記錄見面時間
- 🤖 **AI 場景描述**：GPT-4o 自動分析照片場景，幫你寫備註
- 🎂 **生日提醒**：近期生日自動通知
- ⏰ **聯繫頻率提醒**：太久沒聯絡的朋友，系統會提醒你
- 📊 **互動統計**：一目了然你的社交狀況

## 技術架構

```
Telegram Bot → n8n → FastAPI → PostgreSQL + pgvector
                                    ↓
                              InsightFace (人臉辨識)
                              GPT-4o (場景描述)
```

## Telegram Bot 指令

| 指令 | 功能 |
|------|------|
| 📷 傳照片 | 自動辨識並記錄 |
| `/new 姓名` | 新增聯絡人 |
| `/info 姓名` | 查看資訊 |
| `/birthday 姓名 日期` | 設定生日 |
| `/note 姓名 內容` | 新增備註 |
| `/greet 姓名` | 記錄問候 |
| `/freq 姓名 天數` | 設定聯繫頻率 |
| `/remind` | 需要聯繫的人 |
| `/upcoming` | 近期生日 |
| `/list` | 所有聯絡人 |
| `/history 姓名` | 互動歷史 |
| `/stats` | 統計數據 |
